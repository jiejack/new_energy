# 快速开始

本文档帮助您在5分钟内快速上手新能源监控系统。

## 前置条件

- 系统已成功安装（参考 [安装指南](./Installation-Guide)）
- 已创建管理员账户

## 登录系统

### 1. 访问登录页面

打开浏览器，访问系统地址（默认 `http://localhost`）

### 2. 输入账户信息

| 字段 | 说明 |
|------|------|
| 用户名 | 管理员用户名（默认：admin） |
| 密码 | 管理员密码 |

### 3. 登录成功

登录后将跳转到仪表盘页面，显示系统概览信息。

## 功能导航

系统主要功能模块：

```
┌─────────────────────────────────────────────────────────┐
│  仪表盘  │  实时监控  │  设备管理  │  告警管理  │  数据查询  │  系统设置  │
└─────────────────────────────────────────────────────────┘
```

| 模块 | 功能 | 路径 |
|------|------|------|
| 仪表盘 | 系统概览、关键指标 | /dashboard |
| 实时监控 | 设备实时状态、数据展示 | /monitor |
| 设备管理 | 电站、设备、采集点管理 | /device |
| 告警管理 | 告警列表、规则配置、通知设置 | /alarm |
| 数据查询 | 历史数据、报表生成 | /data |
| 系统设置 | 用户、角色、权限、系统配置 | /system |

## 快速操作

### 添加电站

1. 进入 **设备管理** → **电站管理**
2. 点击 **新建电站** 按钮
3. 填写电站信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| 电站名称 | 电站标识名称 | 光伏电站A |
| 电站类型 | 光伏/风电/储能 | 光伏 |
| 装机容量 | 装机容量(kW) | 1000 |
| 地址 | 电站地址 | 北京市朝阳区 |
| 经纬度 | 地理坐标 | 116.4, 39.9 |

4. 点击 **保存** 完成创建

### 添加设备

1. 进入 **设备管理** → **设备管理**
2. 点击 **新建设备** 按钮
3. 填写设备信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| 设备名称 | 设备标识 | 逆变器#1 |
| 设备类型 | 设备类型 | 逆变器 |
| 所属电站 | 关联电站 | 光伏电站A |
| 通信地址 | 设备地址 | 192.168.1.100 |
| 通信协议 | 通信协议 | Modbus TCP |

4. 点击 **保存** 完成创建

### 配置告警规则

1. 进入 **告警管理** → **告警规则**
2. 点击 **新建规则** 按钮
3. 配置规则：

| 字段 | 说明 | 示例 |
|------|------|------|
| 规则名称 | 规则标识 | 温度超限告警 |
| 规则类型 | 限值/趋势/自定义 | 限值 |
| 告警级别 | 提示/警告/严重/紧急 | 严重 |
| 触发条件 | 触发表达式 | temperature > 80 |
| 持续时间 | 持续时间(秒) | 60 |
| 通知渠道 | 通知方式 | 邮件,短信 |

4. 点击 **保存** 完成配置

### 查看实时数据

1. 进入 **实时监控** → **电站监控**
2. 选择要查看的电站
3. 查看实时数据：
   - 设备状态
   - 功率曲线
   - 发电量统计
   - 环境参数

### 导出数据报表

1. 进入 **数据查询** → **历史数据**
2. 设置查询条件：
   - 时间范围
   - 设备选择
   - 数据类型
3. 点击 **查询** 获取数据
4. 点击 **导出** 选择格式（Excel/CSV）

## API调用示例

### 获取Token

```bash
curl -X POST http://localhost:8080/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "username": "admin",
    "password": "your_password"
  }'
```

响应：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIs...",
    "expire": "2026-04-08T12:00:00Z"
  }
}
```

### 查询电站列表

```bash
curl -X GET http://localhost:8080/api/v1/stations \
  -H "Authorization: Bearer <token>"
```

### 查询设备数据

```bash
curl -X GET "http://localhost:8080/api/v1/devices?station_id=<station_id>" \
  -H "Authorization: Bearer <token>"
```
