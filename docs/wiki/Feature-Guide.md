## 1. 系统概览

新能源监控系统是一套面向光伏电站、风电场、储能站等新能源设施的智能化监控管理平台。系统采用微服务架构，支持大规模设备接入、实时数据采集、智能告警分析和可视化展示。

### 1.1 系统架构

- **前端**：Vue 3 + TypeScript + Element Plus
- **后端**：Go 1.21 + Gin + GORM
- **数据库**：PostgreSQL + Redis + Kafka
- **监控**：Prometheus + Grafana

### 1.2 系统特点

- **实时性**：毫秒级数据采集和处理
- **可靠性**：高可用架构，支持集群部署
- **扩展性**：模块化设计，易于扩展新功能
- **安全性**：完善的权限控制和安全机制
- **智能性**：AI辅助分析和预测

## 2. 核心功能

### 2.1 多协议支持

| 协议 | 版本 | 支持功能 | 应用场景 |
|------|------|----------|----------|
| Modbus | TCP/RTU | 遥测、遥信、遥控 | 光伏逆变器、电表、汇流箱 |
| IEC104 | - | 遥测、遥信、遥控 | 变电站、电网设备 |
| IEC61850 | - | 遥测、遥信、遥控、遥调 | 智能变电站、高压设备 |
| MQTT | 3.1.1/5.0 | 数据采集、设备管理 | 物联网设备、传感器 |

### 2.2 实时监控

- **实时数据采集**：支持毫秒级数据采集
- **设备状态监控**：实时展示设备运行状态
- **功率曲线**：实时功率变化曲线
- **数据看板**：多维度数据可视化

### 2.3 智能告警

- **基于规则的告警**：支持阈值告警、趋势告警、质量告警
- **多渠道通知**：邮件、短信、Webhook
- **告警级别**：紧急、重要、一般、提示
- **告警处理**：确认、清除、升级
- **告警统计**：按级别、类型、设备统计

### 2.4 数据分析

- **历史数据查询**：支持多维度查询
- **数据导出**：Excel、CSV格式
- **趋势分析**：发电趋势、能耗分析
- **异常检测**：基于AI的异常检测
- **预测分析**：发电量预测、设备故障预测

### 2.5 权限管理

- **RBAC模型**：基于角色的权限控制
- **细粒度权限**：菜单权限、操作权限、数据权限
- **用户管理**：用户创建、编辑、禁用
- **角色管理**：角色创建、权限配置
- **操作日志**：完整的操作审计

## 3. 设备管理

### 3.1 区域管理

- **区域层级**：支持多级区域管理
- **区域地图**：地理分布可视化
- **区域统计**：各区域设备数量、发电量统计

### 3.2 电站管理

- **电站信息**：基本信息、技术参数、地理位置
- **电站类型**：光伏、风电、储能、混合
- **装机容量**：总容量、各设备容量
- **电站状态**：运行、维护、停机

### 3.3 设备管理

- **设备类型**：
  - 光伏：逆变器、汇流箱、电表、气象站
  - 风电：风机、塔筒、变桨系统、偏航系统
  - 储能：BMS、PCS、电池簇
  - 电网：变压器、开关柜、保护装置

- **设备参数**：
  - 基本信息：名称、型号、厂家、序列号
  - 技术参数：额定功率、电压等级、电流等级
  - 通信参数：协议、地址、波特率、IP地址

- **设备状态**：
  - 在线/离线
  - 运行/故障/维护
  - 功率状态
  - 温度状态

### 3.4 采集点管理

- **采集点类型**：
  - 遥测：电流、电压、功率、温度等
  - 遥信：开关状态、故障状态等
  - 遥控：远程控制命令
  - 遥调：远程参数调节

- **采集配置**：
  - 寄存器地址
  - 数据类型
  - 单位
  - 采集间隔
  - 报警阈值

## 10. 功能使用流程

### 10.1 设备接入流程

1. **添加区域**：创建电站所属区域
2. **添加电站**：创建电站基本信息
3. **添加设备**：配置设备参数和通信信息
4. **配置采集点**：设置设备采集点和参数
5. **测试连接**：验证设备通信状态
6. **启用采集**：开始数据采集

### 10.2 告警配置流程

1. **创建告警规则**：设置规则名称和类型
2. **配置触发条件**：设置阈值和持续时间
3. **配置通知方式**：选择通知渠道和接收人
4. **测试规则**：验证规则有效性
5. **启用规则**：开始告警监控

### 10.3 报表生成流程

1. **选择报表类型**：日报、月报、年报
2. **设置时间范围**：选择报表周期
3. **选择电站/设备**：选择要包含的对象
4. **配置报表内容**：选择要包含的数据项
5. **生成报表**：系统自动生成报表
6. **导出报表**：下载或分享报表
