## 1. 系统概述

### 1.1 系统简介

新能源监控系统是一套面向光伏电站、风电场、储能站等新能源设施的智能化监控管理平台。系统提供实时数据采集、设备监控、告警管理、数据分析等功能，帮助运维人员高效管理电站运营。

### 1.2 功能模块

| 模块 | 功能说明 |
|------|---------|
| 首页 | 数据概览、实时监控、告警统计 |
| 设备管理 | 区域、电站、设备、采集点管理 |
| 数据监控 | 实时数据、历史数据查询 |
| 告警管理 | 告警列表、告警规则、通知配置 |
| 数据报表 | 日报、月报、年报生成与导出 |
| 系统设置 | 用户管理、角色权限、系统配置 |

### 1.3 浏览器要求

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## 2. 登录与认证

### 2.1 登录系统

1. 打开浏览器，访问系统地址
2. 输入用户名和密码
3. 点击"登录"按钮

![登录界面](./images/login.png)

### 2.2 修改密码

1. 点击右上角用户头像
2. 选择"个人中心"
3. 点击"修改密码"
4. 输入旧密码和新密码
5. 点击"确定"保存

### 2.3 退出登录

1. 点击右上角用户头像
2. 选择"退出登录"

## 3. 系统首页

### 3.1 首页概览

首页展示系统核心数据概览，包括：

- **统计卡片**: 电站数量、设备数量、今日发电量、告警数量
- **实时功率图**: 各电站实时功率曲线
- **电站列表**: 电站状态和关键指标
- **告警列表**: 最新告警信息
- **电站地图**: 电站地理位置分布

### 3.2 数据刷新

- 统计数据每5分钟自动刷新
- 实时功率每5秒自动更新
- 告警列表实时推送

## 4. 设备管理

### 4.1 区域管理

#### 新增区域

1. 进入"设备管理 > 区域管理"
2. 点击"新增"按钮
3. 填写区域信息：
   - 区域名称
   - 区域编码
   - 上级区域（可选）
   - 描述
4. 点击"确定"保存

#### 编辑区域

1. 在区域列表中找到目标区域
2. 点击"编辑"按钮
3. 修改区域信息
4. 点击"确定"保存

#### 删除区域

1. 在区域列表中找到目标区域
2. 点击"删除"按钮
3. 确认删除操作

> **注意**: 删除区域前，请确保该区域下没有子区域和电站。

### 4.2 电站管理

#### 新增电站

1. 进入"设备管理 > 电站管理"
2. 点击"新增"按钮
3. 填写电站信息：
   - 电站编码
   - 电站名称
   - 电站类型（光伏/风电/储能）
   - 所属区域
   - 装机容量
   - 电压等级
   - 经纬度
   - 地址
4. 点击"确定"保存

#### 电站详情

1. 在电站列表中点击电站名称
2. 查看电站详细信息：
   - 基本信息
   - 设备列表
   - 实时数据
   - 历史数据
   - 告警记录

### 4.3 设备管理

#### 新增设备

1. 进入"设备管理 > 设备管理"
2. 点击"新增"按钮
3. 填写设备信息：
   - 设备编码
   - 设备名称
   - 设备类型
   - 所属电站
   - 厂家
   - 型号
   - 序列号
   - 额定参数
   - 通信参数
4. 点击"确定"保存

#### 设备类型说明

| 设备类型 | 说明 |
|---------|------|
| 逆变器 | 光伏逆变器 |
| 电表 | 电能计量表 |
| 汇流箱 | 光伏汇流箱 |
| 变压器 | 电力变压器 |
| 气象站 | 气象监测设备 |
| 风机 | 风力发电机组 |
| BMS | 电池管理系统 |

### 4.4 采集点管理

#### 新增采集点

1. 进入"设备管理 > 采集点管理"
2. 点击"新增"按钮
3. 填写采集点信息：
   - 采集点编码
   - 采集点名称
   - 采集点类型（遥信/遥测/遥控/遥调）
   - 所属设备
   - 数据类型
   - 单位
   - 寄存器地址
4. 点击"确定"保存

## 5. 数据监控

### 5.1 实时监控

#### 查看实时数据

1. 进入"数据监控 > 实时监控"
2. 选择电站和设备
3. 查看实时数据：
   - 功率曲线
   - 发电量统计
   - 设备状态
   - 关键参数

#### 数据刷新

- 点击"刷新"按钮手动刷新
- 支持自动刷新（可设置刷新间隔）

### 5.2 历史数据

#### 查询历史数据

1. 进入"数据监控 > 历史数据"
2. 设置查询条件：
   - 选择电站/设备/采集点
   - 选择时间范围
   - 设置采样间隔
3. 点击"查询"按钮
4. 查看历史数据曲线和表格

#### 导出数据

1. 查询历史数据后
2. 点击"导出"按钮
3. 选择导出格式（Excel/CSV）
4. 下载导出文件

## 6. 告警管理

### 6.1 告警列表

#### 查看告警

1. 进入"告警管理 > 告警列表"
2. 查看告警信息：
   - 告警级别
   - 告警标题
   - 告警内容
   - 触发时间
   - 当前状态

#### 告警筛选

可按以下条件筛选告警：
- 电站
- 设备
- 告警级别
- 告警状态
- 时间范围

#### 确认告警

1. 选择需要确认的告警
2. 点击"确认"按钮
3. 填写确认备注
4. 点击"确定"

#### 清除告警

1. 选择需要清除的告警
2. 点击"清除"按钮
3. 填写清除原因
4. 点击"确定"

### 6.2 告警规则

#### 新增告警规则

1. 进入"告警管理 > 告警规则"
2. 点击"新增"按钮
3. 配置规则信息：
   - 规则名称
   - 规则类型
   - 告警级别
   - 触发条件
   - 阈值设置
   - 持续时间
   - 通知渠道
   - 通知人员
4. 点击"确定"保存

#### 规则类型说明

| 规则类型 | 说明 |
|---------|------|
| 越限告警 | 数据超过阈值触发 |
| 趋势告警 | 数据变化趋势异常触发 |
| 质量告警 | 数据质量异常触发 |
| 通信告警 | 设备通信异常触发 |

#### 测试规则

1. 在规则列表中找到目标规则
2. 点击"测试"按钮
3. 查看测试结果

### 6.3 通知配置

#### 配置邮件通知

1. 进入"告警管理 > 通知配置"
2. 选择"邮件"渠道
3. 点击"配置"按钮
4. 填写邮件服务器信息：
   - SMTP服务器地址
   - SMTP端口
   - 发件人邮箱
   - 邮箱密码/授权码
   - 是否使用SSL
5. 点击"保存"

#### 配置短信通知

1. 选择"短信"渠道
2. 点击"配置"按钮
3. 填写短信服务信息：
   - AccessKey
   - AccessSecret
   - 短信签名
   - 短信模板
4. 点击"保存"

#### 测试通知

1. 配置完成后
2. 点击"测试"按钮
3. 输入测试接收地址
4. 发送测试通知

## 7. 数据报表

### 7.1 生成报表

#### 日报

1. 进入"数据报表"
2. 选择报表类型"日报"
3. 选择日期
4. 选择电站（可选全部）
5. 点击"查询"按钮

#### 月报

1. 选择报表类型"月报"
2. 选择月份
3. 选择电站
4. 点击"查询"按钮

#### 年报

1. 选择报表类型"年报"
2. 选择年份
3. 选择电站
4. 点击"查询"按钮

### 7.2 报表内容

报表包含以下内容：

- **发电量统计**: 总发电量、峰值发电量、平均发电量
- **设备运行统计**: 在线时长、离线时长、可用率
- **告警统计**: 各级别告警数量
- **趋势图表**: 发电量趋势、告警分布

### 7.3 导出报表

1. 生成报表后
2. 点击"导出报表"按钮
3. 选择导出格式（Excel/CSV）
4. 下载报表文件

## 9. 常见问题

### 9.1 登录问题

**Q: 忘记密码怎么办？**

A: 请联系系统管理员重置密码。

**Q: 登录后自动退出？**

A: 可能是Token过期，请重新登录。如果问题持续，请检查浏览器Cookie设置。

### 9.2 数据问题

**Q: 实时数据不更新？**

A: 
1. 检查网络连接
2. 点击刷新按钮
3. 检查设备是否在线

**Q: 历史数据查询不到？**

A: 
1. 检查时间范围是否正确
2. 确认设备在该时间段有数据
3. 联系管理员检查数据存储

### 9.3 告警问题

**Q: 收不到告警通知？**

A: 
1. 检查通知配置是否正确
2. 检查通知渠道是否启用
3. 检查用户是否在通知人员列表中

**Q: 告警规则不生效？**

A: 
1. 检查规则是否启用
2. 检查规则条件是否正确
3. 检查关联的设备/采集点是否正确

### 9.4 系统问题

**Q: 页面加载慢？**

A: 
1. 检查网络连接
2. 清除浏览器缓存
3. 尝试使用其他浏览器

**Q: 导出文件失败？**

A: 
1. 检查浏览器是否阻止下载
2. 检查磁盘空间
3. 尝试减少导出数据量
