#### 3.1.2 processor 包修复

**问题**: 多个滤波器的 logger 字段未初始化

**修复的文件**: [filter.go](file:///workspace/pkg/processor/filter.go)

修复的滤波器:
- MovingAverageFilter
- MedianFilter
- KalmanFilter
- LimitFilter
- ExponentialSmoothingFilter
- MedianFilterOptimized

**结果**: 所有测试通过 ✓

#### 3.1.3 compute/rule 包修复

**问题 1**: TriggerManager 使用了未初始化的 logger

**修复**: 在 [trigger.go](file:///workspace/pkg/compute/rule/trigger.go) 中添加了 logger 检查，使用 zap.NewNop() 作为备选

**问题 2**: logger.Log 在测试环境中可能为 nil

**修复**: 在 [logger.go](file:///workspace/internal/infrastructure/logger/logger.go) 中添加了 init() 函数，默认初始化为 zap.NewNop()

**结果**: TriggerManager 测试通过 ✓
