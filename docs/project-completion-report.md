## 一、项目概述

### 1.1 项目目标

构建一个基于云原生架构的分布式新能源监控平台，实现设备监控、数据采集、智能告警和数据分析功能。

### 1.2 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Go 1.24 + Gin + GORM |
| 前端 | Vue 3 + TypeScript + Element Plus |
| 数据库 | PostgreSQL 15 |
| 缓存 | Redis 7 |
| 消息队列 | Kafka |
| 容器化 | Docker + Kubernetes |
| 监控 | Prometheus + Grafana |
| 追踪 | Jaeger |
