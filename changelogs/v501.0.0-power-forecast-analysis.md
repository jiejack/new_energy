### Analysis Details

#### 光伏发电预测需求
- **核心影响因子**: 辐照度、温度、云量、阴影、面板倾角
- **预测精度要求**: 小时级MAPE < 8%，日级MAPE < 5%
- **时间范围**: 未来0-72小时
- **更新频率**: 每15分钟更新一次

#### 风力发电预测需求
- **核心影响因子**: 风速、风向、湍流强度、空气密度、桨距角
- **预测精度要求**: 小时级MAPE < 10%，日级MAPE < 7%
- **时间范围**: 未来0-72小时
- **更新频率**: 每30分钟更新一次

#### 储能系统预测需求
- **核心影响因子**: SOC、充放电计划、电价、电网需求
- **预测精度要求**: 小时级MAPE < 5%
- **时间范围**: 未来0-24小时
- **更新频率**: 每小时更新一次
