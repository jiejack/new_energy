## 📡 预测API接口

### 1. 实时预测接口

#### POST /api/v1/ai/predictions

**功能**: 执行单次实时预测

**请求示例**:
```http
POST /api/v1/ai/predictions
Content-Type: application/json
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

{
  "model_id": "solar_forecast_v1.0",
  "version": "latest",
  "station_id": "station_001",
  "device_id": "inverter_001",
  "inputs": {
    "irradiance": 800.5,
    "temperature": 25.3,
    "humidity": 60.2,
    "wind_speed": 2.5
  },
  "options": {
    "include_confidence": true,
    "include_explanation": true,
    "cache_ttl_seconds": 3600
  }
}
```

**成功响应 (200 OK)**:
```json

    "device_id": "inverter_001",

      "value": 4500.5,
      "unit": "kW",
      "confidence": 0.92,
      "confidence_interval": [4200.3, 4800.7]
    },
    "explanation": {
      "feature_importance": {
        "irradiance": 0.65,
        "temperature": 0.20,
        "humidity": 0.10,
        "wind_speed": 0.05
      },
      "shap_values": {
        "irradiance": 1200.5,
        "temperature": -50.2,
        "humidity": 25.1,
        "wind_speed": -12.3
      }
    },

    }

}
```


### 2. 批量预测接口

#### POST /api/v1/ai/predictions/batch

**功能**: 提交批量预测任务

**请求示例**:
```http
POST /api/v1/ai/predictions/batch
Content-Type: application/json

{
  "model_id": "solar_forecast_v1.0",
  "version": "latest",
  "batch_id": "batch_daily_20260415",
  "station_id": "station_001",
  "inputs": [
    {
      "timestamp": "2026-04-15T00:00:00Z",
      "features": { "irradiance": 0, "temperature": 15.0 }
    },
    {
      "timestamp": "2026-04-15T01:00:00Z",
      "features": { "irradiance": 0, "temperature": 14.5 }
    }
  ],
  "callback_url": "https://api.example.com/webhooks/batch-prediction",
  "options": {
    "include_confidence": true,
    "priority": 1
  }
}
```

**成功响应 (202 Accepted)**:
```json
{
  "success": true,
  "data": {
    "job_id": "job_xyz789",
    "status": "queued",
    "estimated_time_seconds": 120,
    "queue_position": 5,
    "total_items": 24
  },
  "message": "Batch prediction job submitted successfully"
}
```

#### GET /api/v1/ai/predictions/batch/{job_id}

**功能**: 查询批量预测任务状态

**响应示例**:
```json
{
  "success": true,
  "data": {
    "job_id": "job_xyz789",
    "batch_id": "batch_daily_20260415",
    "model_id": "solar_forecast_v1.0",
    "status": "running",
    "progress": 45,
    "total_items": 24,
    "processed_items": 11,
    "failed_items": 0,
    "started_at": "2026-04-15T10:00:00Z",
    "estimated_completion_at": "2026-04-15T10:02:00Z"
  }
}
```

#### GET /api/v1/ai/predictions/batch/{job_id}/results

**功能**: 获取批量预测结果

**查询参数**:
- page: 页码 (默认1)
- page_size: 每页数量 (默认100, 最大1000)
- status: 过滤状态 (success/failed/all)

## 📡 WebSocket实时接口

### 连接建立

```
wss://api.example.com/ws/ai?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

### 订阅消息格式

```json
{
  "type": "subscribe",
  "channel": "predictions",
  "filter": {
    "station_id": "station_001"
  }
}
```

### 推送消息类型

| 消息类型 | 说明 |
|---------|------|
| prediction | 新预测结果 |
| alarm | 新告警触发 |
| batch_job_update | 批量任务状态更新 |
| model_update | 模型更新通知 |

### 预测结果推送示例

```json
{
  "type": "prediction",
  "timestamp": "2026-04-15T10:00:00Z",
  "payload": {
    "prediction_id": "pred_001",
    "model_id": "solar_forecast_v1.0",
    "station_id": "station_001",
    "target_time": "2026-04-15T11:00:00Z",
    "prediction_value": 4500.5,
    "confidence": 0.92
  }
}
```

## 🔐 错误码定义

| 错误码 | HTTP状态码 | 说明 |
|-------|-----------|------|
| MODEL_NOT_FOUND | 404 | 模型不存在 |
| MODEL_DISABLED | 400 | 模型已禁用 |
| INVALID_INPUT | 400 | 输入参数无效 |
| MISSING_FEATURE | 400 | 缺少必需特征 |
| INFERENCE_ERROR | 500 | 推理执行错误 |
| JOB_NOT_FOUND | 404 | 任务不存在 |
| RATE_LIMIT_EXCEEDED | 429 | 超出调用频率限制 |
| UNAUTHORIZED | 401 | 未授权访问 |
| FORBIDDEN | 403 | 无权限访问 |
| INTERNAL_ERROR | 500 | 内部服务器错误 |
