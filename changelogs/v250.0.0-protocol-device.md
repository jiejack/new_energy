# [250.0.0] - 2026-04-15

## 🔌 Protocol & Device Support

### Added
- **Protocol Support**
  - Complete IEC 61850 MMS implementation with all ACSI services
  - Modbus TCP/RTU/ASCII with advanced features (function codes 1-127)
  - IEC 60870-5-101/104 with balanced and unbalanced modes
  - DNP3 with secure authentication and time synchronization
  - OPC UA with full information model and pub/sub

- **Device Support**
  - 500+ device drivers from major manufacturers (Siemens, ABB, Schneider)
  - Auto-discovery for all supported protocols
  - Device profile management with templates
  - Firmware update over-the-air (OTA)
  - Remote configuration and diagnostics

- **Edge Computing**
  - Edge gateway software for field deployment
  - Local data processing and caching
  - Offline operation with automatic sync
  - Edge analytics and rule execution
  - Edge device management and monitoring

### Changed
- Rewrote protocol stack for better performance and reliability
- Implemented plugin architecture for custom protocols
- Optimized message parsing with zero-copy techniques
- Updated device communication library to latest versions

### Fixed
- Connection issues with certain device models
- Timeout handling in slow network conditions
- Data corruption in protocol message parsing
- Memory leaks in long-running connections
