# 常见错误解决方案

## 后端常见错误

### 1. 数据库连接错误
- **错误信息**：`dial tcp 127.0.0.1:5432: connection refused`
- **解决方案**：
  - 检查数据库服务是否启动
  - 检查数据库连接配置是否正确
  - 检查网络连接是否正常

### 2. 依赖注入错误
- **错误信息**：`missing parameter: ...`
- **解决方案**：
  - 运行 `go run github.com/google/wire/cmd/wire` 重新生成 wire_gen.go
  - 检查依赖注入配置是否正确

### 3. API 路由错误
- **错误信息**：`404 Not Found`
- **解决方案**：
  - 检查 API 路径是否正确
  - 检查路由注册是否完整
  - 检查 HTTP 方法是否正确

## 前端常见错误

### 1. TypeScript 类型错误
- **错误信息**：`Argument of type 'number' is not assignable to parameter of type 'string'`
- **解决方案**：
  - 检查变量类型是否匹配
  - 使用类型断言或类型转换
  - 修正类型定义

### 2. API 调用错误
- **错误信息**：`Network Error`
- **解决方案**：
  - 检查网络连接
  - 检查 API 地址是否正确
  - 检查后端服务是否启动

### 3. 组件渲染错误
- **错误信息**：`Cannot read properties of undefined`
- **解决方案**：
  - 检查数据是否正确加载
  - 使用可选链操作符 `?.`
  - 添加数据加载状态检查

## 构建错误

### 1. 前端构建错误
- **错误信息**：`vue-tsc -b failed`
- **解决方案**：
  - 修复 TypeScript 类型错误
  - 检查依赖是否安装正确
  - 清除缓存后重新构建

### 2. 后端编译错误
- **错误信息**：`undefined: math`
- **解决方案**：
  - 检查导入是否正确
  - 检查依赖是否安装
  - 检查代码语法是否正确

## 部署错误

### 1. 端口占用错误
- **错误信息**：`listen tcp :8080: bind: address already in use`
- **解决方案**：
  - 检查端口是否被占用
  - 修改配置使用其他端口
  - 停止占用端口的进程

### 2. 环境变量错误
- **错误信息**：`configPath is empty`
- **解决方案**：
  - 检查环境变量是否设置
  - 提供默认配置文件路径
  - 检查配置文件是否存在
