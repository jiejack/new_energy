# 报表功能说明

## 功能概述

统计报表功能是新能源监控系统的核心功能之一，主要用于对电站运行数据进行统计分析和可视化展示，为管理层提供决策支持。

## 报表类型

### 1. 日报表
- **统计周期**：每日
- **数据内容**：
  - 各电站当日发电量
  - 各电站当日告警数
  - 各电站当日在线率
  - 当日总发电量
  - 当日总告警数
  - 当日平均在线率

### 2. 周报表
- **统计周期**：每周
- **数据内容**：
  - 各电站周发电量
  - 各电站周告警数
  - 各电站周在线率
  - 周总发电量
  - 周总告警数
  - 周平均在线率
  - 与上周数据的环比分析

### 3. 月报表
- **统计周期**：每月
- **数据内容**：
  - 各电站月发电量
  - 各电站月告警数
  - 各电站月在线率
  - 月总发电量
  - 月总告警数
  - 月平均在线率
  - 与上月数据的环比分析
  - 与去年同期的同比分析

### 4. 年报表
- **统计周期**：每年
- **数据内容**：
  - 各电站年发电量
  - 各电站年告警数
  - 各电站年在线率
  - 年总发电量
  - 年总告警数
  - 年平均在线率
  - 与去年数据的同比分析

## 报表功能

### 1. 数据查询
- 支持按时间范围查询
- 支持按电站筛选
- 支持按报表类型选择

### 2. 数据展示
- 表格展示详细数据
- 图表展示趋势和分布
- 支持数据排序和筛选

### 3. 数据导出
- 支持 Excel 格式导出
- 支持 CSV 格式导出
- 导出文件包含完整数据和汇总信息

### 4. 数据分析
- 环比分析（与上一周期比较）
- 同比分析（与去年同期比较）
- 趋势分析（历史数据趋势）

## 技术实现

- **后端**：使用 Go 语言实现，基于 Gin 框架
- **前端**：使用 Vue 3 + TypeScript 实现
- **数据存储**：使用关系型数据库存储基础数据，使用时序数据库存储历史数据
- **报表生成**：使用 excelize 库生成 Excel 报表
- **图表展示**：使用 ECharts 库实现数据可视化
