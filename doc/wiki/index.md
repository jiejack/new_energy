# 新能源监控系统知识库

## 项目概述

本知识库是为新能源监控系统项目服务的专业知识库，采用LLM-wiki架构模式，实现知识的持续积累、自动交叉引用和主动检测矛盾。经过500轮迭代，知识库已形成完整的知识体系。
